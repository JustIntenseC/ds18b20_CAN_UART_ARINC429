# DS18B20 Controller (1921ВК035)

**Описание проекта**
Этот проект представляет собой контроллер для работы с датчиками температуры **DS18B20** на базе микроконтроллера **1921ВК035**. Проект разработан с нуля, включая драйвер для взаимодействия с датчиками. Поддерживаются два интерфейса передачи данных: **UART** и **CAN**. Реализованы режимы опроса датчиков: **ручной**, **автоматический** и **неблокирующий**.

**Примечание:** Опрос датчиков в неблокирующем режиме позволяет избежать нагрузки на процессор, обеспечивая эффективную многозадачность.

---

## Основные возможности
- **Поддержка интерфейсов**: UART и CAN для передачи данных.
- **Режимы опроса датчиков**:
  - **Ручной режим**: опрос датчиков по команде пользователя.
  - **Автоматический режим**: периодический опрос датчиков с заданным интервалом.
  - **Неблокирующий режим**: опрос датчиков без блокировки основного потока выполнения, что минимизирует нагрузку на процессор.
- **Драйвер DS18B20**: разработан с нуля для обеспечения стабильной работы с датчиками.
- **Гибкая конфигурация**: возможность настройки параметров опроса и передачи данных.

## Настройка и запуск

### 1. Подключение датчиков
- Подключите датчики **DS18B20** к соответствующим выводам микроконтроллера **1921ВК035**.
- Убедитесь, что линии данных подтянуты к питанию через резистор **4.7 кОм**.

### 2. Конфигурация
- Настройте параметры в файле конфигурации:
  - Скорость передачи данных для **UART** и **CAN**.
  - Интервал опроса датчиков в автоматическом режиме.
  - Адреса датчиков (если используются несколько датчиков).

### 3. Компиляция и прошивка
- Скомпилируйте проект с помощью вашей среды разработки.
- Прошейте микроконтроллер **1921ВК035** полученным файлом.
